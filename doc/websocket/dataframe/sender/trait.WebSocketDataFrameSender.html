A trait that allows for sending data frames.

A data frame sender contains an inner Writer, on to which data frames are written. Data frame senders convert data frames into raw bytes to be written to a Writer. The function RawDataFrame.write() writes RawDataFrames to a Writer, allowing the data frame sender to simply convert a WebSocketDataFrame into a RawDataFrame and let the RawDataFrame do the writing.
