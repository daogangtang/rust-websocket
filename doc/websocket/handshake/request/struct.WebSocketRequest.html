Represents a WebSocket request

A WebSocketRequest is used to either make a connection to a WebSocket server using connect() or connect_with(), or received by a server and processed using read() and accept().

A type parameter that is either Inbound or Outbound ensures that only the appropriate methods can be called on either kind of request.

The headers field allows access to the HTTP headers in the request, but short-cut methods are available for accessing common headers.
